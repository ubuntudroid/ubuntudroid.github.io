--- layout: post title: Adding CXF runtime environment to Eclipse date: '2012-04-13T16:53:00.002+02:00' author: Sven Bendel tags: - Java - CXF - JAX-WS - Eclipse modified_time: '2012-04-13T16:53:50.908+02:00' blogger_id: tag:blogger.com,1999:blog-932652460086490809.post-8264413955597269671 blogger_orig_url: http://ubuntudroid.blogspot.com/2012/04/adding-cxf-runtime-environment-to.html --- Another quick tip for JAX-WS development in Eclipse:

When adding a CXF-Runtime via Window->Preferences->CXF 2.x Preferences don't forget to set a checkmark to the left of the entry, otherwise Eclipse won't use the environment and will complain about a missing CXF jar.
